# Skills & Resource Management System

## Project Overview
A full-stack internal management system for a small consultancy or tech agency to manage personnel, track skills and proficiency levels, manage projects, and automatically match personnel to project requirements.

## Tech Stack
- **Frontend**: React.js, Tailwind CSS, Axios
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Architecture**: REST APIs, MVC pattern

## Prerequisites
- Node.js (v16+)
- MySQL (v8+)
- npm or yarn
