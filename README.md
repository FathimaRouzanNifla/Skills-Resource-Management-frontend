## **Skills Management System - Frontend**

This is the **React frontend** for the Skills Management System.  
It interacts with the backend API to manage **personnel, skills, and projects**.

## **1. Features**

- View and manage **personnel**  
- View and manage **skills**  
- View and manage **projects**  
- Assign skills to personnel  
- Assign required skills to projects  
- Responsive UI using **Tailwind CSS**

---

## **2. Tech Stack**

- React.js  
- Tailwind CSS  
- Axios  
- React Router  
- Lucide-React Icons  
- Vite (Build Tool)

---

## **3. Prerequisites**

- Node.js (https://nodejs.org)  
- npm  
- Backend API running at http://localhost:5000

---

## **4. Installation**

Clone the repository:

    git clone https://github.com/FathimaRouzanNifla/Skills-Resource-Management-frontend.git
    cd Skills-Resource-Management-frontend

Install dependencies:

    npm install

---

## **5. Environment Variables**

Create a .env file in the frontend root directory and add:

    VITE_API_BASE_URL=http://localhost:5000/api

---

## **6. Running Frontend**

Start the development server:

    npm run dev

Application will run at:

    http://localhost:5173

---

## **7. Folder Structure**

    src/
