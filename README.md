## **Skills Management System - Frontend**

This is the **React frontend** for the Skills Management System.  
It interacts with the backend API to manage **personnel, skills, and projects**.

## **1. Features**

- View and manage **personnel**  
- View and manage **skills**  
- View and manage **projects**  
- Assign skills to personnel  
- Assign required skills to projects  
- Responsive UI using **Tailwind CSS**  

---

## **2. Tech Stack**

- React.js  
- Tailwind CSS  
- Axios  
- React Router  
- Lucide-React Icons  
- Vite (Build Tool)  

---

## **3. Prerequisites**

- Node.js ([https://nodejs.org](https://nodejs.org))  
- npm  
- Backend API running (`http://localhost:5000`)
